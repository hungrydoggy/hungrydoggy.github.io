장바구니에 넣기 버튼을 backend와 연동해 주세요.  

&nbsp;  
### 목표
- 로딩동그라미를 띄운다
- ServerApi에, createCustomerHasItem(int customer_id, int item_id) 함수를 만들어 사용한다
- api 호출 후에 에러가 없으면, 장바구니 위치로 이동해야 한다
- 장바구니에 실제로 추가가 되어 있어야 한다
